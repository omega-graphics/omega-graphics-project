======================
Omega Graphics Project
======================

Omega Graphics Project is a multi-module native graphics stack covering shared runtime utilities, rendering, UI, build tooling, and engine work.

This repo-level site is the suite overview for the generated documentation set. Use the global navigation bar to move between the shared docs and the module-specific Sphinx sites, and use the generated landing page in the output root to jump into either the Sphinx guides or the Doxygen API references.
