======================
Omega Graphics Project
======================

Omega Graphics Project is a collection modular technologies for developing next-gen graphics applications.
Whether this be UI frameworks, game engines, or AutoCAD software, this project provides all the tools and libraries you will ever need.

This repo-level site is the suite overview for the generated documentation set. 
Use the global navigation bar to move between the shared docs and the module-specific Sphinx sites, and use the generated landing page in the output root to jump into either the Sphinx guides or the Doxygen API references.
