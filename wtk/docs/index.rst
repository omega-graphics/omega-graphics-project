========
OmegaWTK
========

This site features all the documentation for using OmegaWTK.

The API reference is split across three documents that mirror the
layout of the headers: the **Composition** layer (drawing primitives,
brushes, animation timing), the **UI** layer (application shell,
windows, views, the style cascade, menus, dialogs, media), and the
**Widgets** layer (the high-level component library that most
applications build directly against).
